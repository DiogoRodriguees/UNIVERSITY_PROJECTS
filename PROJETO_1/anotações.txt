* Ideia inicial
Criar classes para PROFESSORES, ALUNOS e MONITORES para 
melhorar o controle da ações que cada um pode realizar.

Utilizar sinais(SIGINT, SIGTERM) para que o professor feche a sala. 

Enquanto a sala estiver aberta realizar comportamento de entrada e 
saida da sala de maneira aleatório.

* Não feito
1. Verificar se ao sair o monitor mantem a condição 1 proposta pela exercicio
2. Ao entra na sala o monitor deve liberar uma 
   quantidade de tokens que mantenha a principio 1.
   Exemplo: Se houver 5 alunos na sala e o limite é 10,
   quando o novo monitor entrar ele deve liberar apenas mais 5 tokens
3. Garantir que a sala seja fechada corretamente

* Duvidas
1. Pode ter mais de um monitor para cada X alunos.
2. O monitor conta para a quantidade de pessoas na sala?    
3. Pode haver mais de um monitor na sala?
4. Pode haver mais mais de uma grupo?
5. Descrição dever detalhar as ações que ocorrem no sistema?
   
* Funcionalidades
1. Deve haver 1 monitor para cada X alunos.
2. Os alunos so podem entrar se um monitor estiver na sala.
3. O monitor so pode sair se a condição 1 for mantida.
4. Quando a sala é fechada os alunos e monitores não podem entrar.
